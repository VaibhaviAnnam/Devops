Q1) Create AWS instance & connect it to PuTTY
Step 1: Open AWS
Step 2: EC2 → Launch Instance → Name it
Step 3: Create new key pair → Name it
        Key pair type: RSA
        Private key file format: .ppk
        (Download the .ppk key)
Step 4: Network settings → allow SSH, HTTP, HTTPS
Step 5: Then click Launch Instance
Step 6: Open PuTTY
Step 7: For Host Name → click on the instance created on AWS, copy the IPv4 address, and paste it here.
Step 8: Connection type: SSH
Step 8: On the left-hand side, under Connection → SSH, click on Auth
Step 9: Click on Browse → select the downloaded .ppk key
Step 10: Then click Open
Step 11: A terminal will open → login as: ec2-user
         It will show something like: ec2-user@ip

Q8) Build pipeline project in Jenkins
Step 1: Dashboard → New Item → Enter name → pipeline → save
Step 2: Go to Pipeline created 
        Scroll down to definition → under Pipeline script → 
        → Big text (paste the sample code )

pipeline {
    agent any

    stages {
        stage('Build') {
            steps {
                echo 'Building Hello World project...'
            }
        }

        stage('Test') {
            steps {
                echo 'Running tests...'
            }
        }

        stage('Deploy') {
            steps {
                echo 'Deploying application...'
                echo 'Hello World!'
            }
        }
    }
}

Step 3: Apply & Save
